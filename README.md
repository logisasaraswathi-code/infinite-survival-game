# 🔥 Is It Infinite? - Survival Edition

An interactive multi-level survival math game designed for Maths Expo.

## 🎮 Features

- Randomized questions
- No repeating questions per level
- Automatic level progression
- Survival mode (3 lives)
- Timer per question
- +10 / -10 scoring system
- Explanation popup (Expo version)
- Animated gradient background
- Mobile responsive design
- Browser-based leaderboard

## 🧠 Educational Objective

This project helps students understand:
- Finite vs Infinite sets
- Countable and uncountable infinity
- Real-world mathematical reasoning

## 🚀 Technologies Used

- HTML
- CSS (Animated UI)
- JavaScript (Game Logic)
- LocalStorage (Leaderboard)

## 📱 Compatible With

- Desktop
- Mobile
- Tablet

## 🌍 Deployment

Hosted using Render / GitHub Pages.
